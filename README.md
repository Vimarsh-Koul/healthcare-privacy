# HealthCare — Privacy Policy

Public privacy policy for the **HealthCare** app's WHOOP integration (the application
source code is kept in a separate private repository).

- **Markdown:** [`PRIVACY.md`](PRIVACY.md)
- **Hosted page:** `index.html` (enable GitHub Pages → Settings → Pages → deploy from `main` / root)

Once Pages is enabled, the policy is served at
`https://<your-username>.github.io/healthcare-privacy/` — use that URL in the WHOOP
developer dashboard's "Link to your privacy policy" field.
